## For NodeJS / N8N users

For a custom n8n node check <a href="https://www.npmjs.com/package/n8n-nodes-growchief" target="_blank">GrowChief n8n custom node</a><br />

## Public API allows you to work with GrowChief

To use the Public API, go to GrowChief settings and copy the API Key.<br />
With each request you need to pass the API Key in the header `Authorization: {apiKey}`.<br />
If you are using the hosted version of GrowChief the URL will be `https://api.growchief.com/public`.<br />
If you are self-hosting GrowChief the URL will be `https://{NEXT_PUBLIC_BACKEND_URL}/api/public`.<br />

<br />

Here are the following API endpoints you can use:

### Get all workflows

```http
GET https://api.growchief.com/public/workflows
```

Response:

```json
[
  {
    "id": "fbff241-607c-4432-b76d-bdc7bccd5fc3",
    "name": "New Workflow",
    "organizationId": "1b5ca8a2-c71e-4887-8e43-1ddcbca2f1f9",
    "active": true,
    "createdAt": "2025-08-19T07:27:13.902Z",
    "updatedAt": "2025-08-26T09:54:34.720Z",
    "deletedAt": null
  }
]
```

### Add lead to a workflow

```http
POST https://api.growchief.com/public/workflows/:id
```
Params:
`id` - workflow id you want to add the lead to (get it from the previous endpoint)


Payload can work can have multiple options: <br />

<strong>Direct URL (100% success):</strong>

```json
{
  "urls": ["https://www.linkedin.com/in/nevo-david"]
}
```

<br /><br />
<strong>Organization Name, First Name, Last Name</strong>

```json
{
  "organization_name": "GrowChief",
  "firstName": "Nevo",
  "lastName": "David"
}
```
<br /><br />
<strong>Email</strong>

```json
{
  "email": "nevo@growchief.com"
}
```

<br /><br />
<strong>Response:</strong>

```json
[
  {
    "status": "success",
    "message": "Workflow started successfully"
  }
]
```
